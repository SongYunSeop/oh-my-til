Get started in seconds

Standalone CLI

No Obsidian needed. Just Node.js and Claude Code.

# Initialize
npx oh-my-til init ~/my-til

# Start learning
cd ~/my-til && claude

# (Optional) MCP server
npx oh-my-til serve ~/my-til
