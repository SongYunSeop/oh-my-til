독립 CLI

Obsidian 없이 사용. Node.js와 Claude Code만 있으면 됩니다.

# 초기화
npx oh-my-til init ~/my-til

# 학습 시작
cd ~/my-til && claude

# (선택) MCP 서버
npx oh-my-til serve ~/my-til

Obsidian 플러그인

임베디드 터미널 + 파일 감시 + 대시보드.

# 플러그인 자동 설치
npx oh-my-til init ~/vault
