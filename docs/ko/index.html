배움을 위한 완벽한 도구

AI 기반 학습
대화로 배웁니다. Claude가 리서치하고, 가르치고, 퀴즈를 내고, 배운 내용을 TIL로 저장합니다.
간격 반복 복습 (SRS)
SM-2 알고리즘으로 TIL 복습 스케줄링. /til-review로 Anki 스타일 평가.
학습 대시보드
TIL 통계, 카테고리 분석, 스트릭, 히트맵을 한눈에 확인.
내장 MCP 서버
Claude Code가 HTTP로 TIL 노트에 직접 접근. 컨텍스트 기반 리서치와 학습.
정적 사이트 배포
단일 명령으로 TIL을 정적 사이트로 배포. 예시 보기
Obsidian 연동
임베디드 터미널, 파일 감시, 대시보드와 함께 Obsidian에서 사용.
🎬

실제 동작 화면

Claude Code가 주제를 리서치하고, 대화형 학습을 안내하며, 구조화된 TIL 노트를 저장합니다.

❯
🔄

학습 루프

두 가지 진입점, 하나의 순환 사이클.

1
/research topic
어떤 주제든 구조화된 학습 백로그를 생성
→
2
/til topic
리서치, 대화형 학습, TIL 노트 저장
→
3
/til-review
SM-2 스케줄링 기반 간격 반복 복습
→
4
/backlog
진행률 확인 및 다음 주제 선택
♻️
사이클이 반복됩니다: /backlog에서 /til로, 완료된 TIL은 /til-review로 스케줄링, /research로 백로그를 계속 채웁니다.
🚀

몇 초 만에 시작하기

Claude Code Plugin
스킬, MCP 서버, 훅이 자동으로 등록됩니다. 권장 방법.
# TIL 디렉토리에서 실행
claude plugin marketplace add SongYunSeop/oh-my-til

# 플러그인 설치
claude plugin install oh-my-til --scope project
클릭하여 복사
Obsidian 연동
선택 사항. Obsidian에 임베디드 터미널과 대시보드를 추가합니다.
# Claude Code 내에서 실행
/setup-obsidian
클릭하여 복사
필요 환경: Node.js 18+ · Claude Code CLI
